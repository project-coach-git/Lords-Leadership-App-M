## ‚ö° Deployment Instructions (Vercel)

This project is optimized for a zero-config deployment on [Vercel](https://vercel.com).

### 1. Create a Repository
1. Create a new **Private** repository on GitHub named `lords-leadership`.
2. Upload the following files to the root:
   - `App.tsx`
   - `liveClient.ts`
   - `index.tsx`
   - `index.html`
   - `metadata.json`

### 2. Deploy to Vercel
1. Log in to Vercel and click **Add New > Project**.
2. Select your `lords-leadership` repository.
3. **IMPORTANT:** Before clicking "Deploy", open the **Environment Variables** section.
4. Add the following key:
   - **Key:** `API_KEY`
   - **Value:** `[Your Google Gemini API Key]`
5. Click **Deploy**.

### 3. Permissions
Ensure you grant **Microphone Access** when prompted by your browser to use the AI Voice Coaching feature.
